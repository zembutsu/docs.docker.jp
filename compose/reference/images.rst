=======================================
docker-compose images
=======================================

.. ```
   Usage: images [options] [SERVICE...]

   Options:
       -q, --quiet  Only display IDs
   ```
::

   利用方法: images [オプション] [SERVICE...]

   オプション:
       -q, --quiet  ID のみを表示します。

.. List images used by the created containers.

生成済コンテナによって利用されているイメージの一覧を表示します。
