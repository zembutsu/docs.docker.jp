.. docker compose kill

=======================================
docker compose kill
=======================================

.. sidebar:: 目次

   .. contents:: 
       :depth: 3
       :local:

.. _compose_kill-description:

説明
==========

.. Force stop service containers.

サービスコンテナを強制停止します。

.. _compose_kill-usage:

使い方
==========

.. code-block:: bash

   $ docker compose kill [options] [SERVICE...]

.. Extended description

.. _compose_kill-extended-description:

補足説明
==========

.. Forces running containers to stop by sending a SIGKILL signal. Optionally the signal can be passed, for example:

実行中のコンテナに ``SIGKILL`` シグナルを送信し、強制停止します。次のように、オプションで各シグナルを渡せます。

.. code-block:: bash

   $ docker-compose kill -s SIGINT
